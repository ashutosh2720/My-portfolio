The flex container properties are: COPY COPY //The flex container properties are: * flex-direction * flex-wrap * flex-flow * justify-content * align-items * align-content The flex-direction Property The flex-direction property defines in which direction the container wants to stack the flex items. Example The column value stacks the flex items vertically (from top to bottom): COPY COPY .flex-container { display: flex; flex-direction: column; } [img](image.png The column-reverse value stacks the flex items vertically (but from bottom to top): COPY COPY .flex-container { display: flex; flex-direction: column-reverse; } [img](image.png The row value stacks the flex items horizontally (from left to right): COPY COPY .flex-container { display: flex; flex-direction: row; } image.png The row-reverse value stacks the flex items horizontally (but from right to left): COPY COPY .flex-container { display: flex; flex-direction: row-reverse; } image.png The flex-wrap Property The flex-wrap property specifies whether the flex items should wrap or not. The examples below have 12 flex items, to better demonstrate the flex-wrap property. Example The wrap value specifies that the flex items will wrap if necessary: COPY COPY .flex-container { display: flex; flex-wrap: wrap; } image.png Example The nowrap value specifies that the flex items will not wrap (this is default): COPY COPY .flex-container { display: flex; flex-wrap: nowrap; } image.png Example The wrap-reverse value specifies that the flexible items will wrap if necessary, in reverse order: COPY COPY .flex-container { display: flex; flex-wrap: wrap-reverse; } image.png The flex-flow Property The flex-flow property is a shorthand property for setting both the flex-direction and flex-wrap properties. Example COPY COPY .flex-container { display: flex; flex-flow: row wrap; } The justify-content Property The justify-content property is used to align the flex items: Example The center value aligns the flex items at the center of the container: COPY COPY .flex-container { display: flex; justify-content: center; } image.png Example Example The flex-start value aligns the flex items at the beginning of the container (this is default): COPY COPY .flex-container { display: flex; justify-content: flex-start; } image.png Example The flex-end value aligns the flex items at the end of the container: COPY COPY .flex-container { display: flex; justify-content: flex-end; } image.pngExample The space-around value displays the flex items with space before, between, and after the lines: COPY COPY .flex-container { display: flex; justify-content: space-around; } image.png Example The space-between value displays the flex items with space between the lines: COPY COPY .flex-container { display: flex; justify-content: space-between; } image.png The align-items Property The align-items property is used to align the flex items.
